# 📊 Modelo de Regressão Gama aplicado a Dados Globais de Energia Sustentável

## 🔎 Visão Geral

Este projeto apresenta a aplicação prática de um **Modelo Linear Generalizado (GLM) com distribuição Gamma** utilizando o dataset *Global Data on Sustainable Energy (2000–2020)* (Kaggle).

A análise tem como foco modelar uma variável energética contínua e positiva a partir de indicadores econômicos e energéticos de aproximadamente 175 países ao longo de 21 anos (2000–2020).

O projeto enfatiza boas práticas em:
- preparação e limpeza de dados
- análise exploratória
- modelagem estatística aplicada
- avaliação e interpretação de resultados

---

## 🎯 Objetivo do Projeto

Aplicar regressão Gamma para:

- Modelar uma variável eficiência energética contínua;
- Identificar variáveis socioeconômicas associadas ao comportamento da variável resposta;
- Avaliar a qualidade do ajuste do modelo;
- Demonstrar aplicação prática de GLMs em dados reais.

---

## ⚙️ Pipeline do Projeto

### 1️⃣ Preparação dos Dados
- Importação da base
- Tratamento de valores ausentes
- Seleção de variáveis relevantes
- Organização no formato país-ano

### 2️⃣ Análise Exploratória
- Estatísticas descritivas
- Análise da distribuição da variável resposta
- Avaliação de correlações

### 3️⃣ Modelagem
- Definição da variável resposta
- Seleção das covariáveis
- Ajuste do GLM com família Gamma (link log)
- Estimação por máxima verossimilhança

### 4️⃣ Avaliação
- Significância estatística das variáveis
- Critério de Informação de Akaike (AIC)
- Diagnóstico de resíduos
- Interpretação dos efeitos estimados

---

## 📊 Principais Resultados

- Identificação das variáveis com maior associação com a variável resposta;
- Modelo com bom ajuste segundo critérios de informação;
- Evidência de relações consistentes entre indicadores energéticos e econômicos.
